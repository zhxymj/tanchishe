# 成员代码分发目录

这个目录用于把代码按 5 名成员的分工单独分开，方便发给组员、写报告或说明个人贡献。

正式可编译项目仍然在仓库根目录和 `src/` 目录中。`member_code/` 里的文件是按分工复制出来的参考副本，组员修改后应把代码合并回 `src/` 对应文件，再统一编译测试。

## 分组

| 成员 | 文件夹 | 负责内容 |
| --- | --- | --- |
| 成员1 | `member1_main_game` | 程序入口、主循环、中文窗口标题、中文状态和难度文本、游戏状态、暂停重开、本局用时、分数速度、最高分和纪录重置 |
| 成员2 | `member2_snake` | 蛇身数据、方向控制、移动、增长、撞墙和撞自身 |
| 成员3 | `member3_food` | 食物生成、糖果色强发光、食物动画、吃食物反馈、粒子和光圈扩散效果 |
| 成员4 | `member4_ui_visual` | 中文字体加载、蓝紫明亮竞技场视觉层级、弱化蓝色棋盘边界、连贯圆形蛇身、信息面板、菜单和弹窗绘制 |
| 成员5 | `member5_controls_build_docs` | 中文按钮文案、主操作和次级操作按钮布局、鼠标交互、按钮动画、构建脚本和说明文档 |

## 合并方式

1. 每个人只修改自己文件夹里的代码。
2. 修改完成后，把对应文件复制回 `src/` 或项目根目录的同名文件。
3. 在项目根目录运行：

```powershell
.\build.bat
```

4. 编译和运行都正常后再提交到 GitHub。
