# 成员4：UI 绘制和视觉表现

## 负责文件

- `ui.c`
- `ui.h`

## 主要任务

- 绘制蓝紫明亮竞技场背景、网格纹理和休闲竞技视觉层级，避免大面积绿色。
- 加载中文字体，保证中文标题、按钮、状态和弹窗正常显示。
- 绘制蓝色棋盘、弱化边界、柔和边缘光和细腻格子。
- 绘制连贯圆形蛇身、连接段、蛇头眼睛和方向表现，让视觉更接近休闲竞技游戏。
- 绘制包含本局时间、速度、等级和最高分的轻量玻璃感信息面板。
- 绘制开始菜单、暂停弹窗和游戏结束弹窗。
- 保证文字、按钮、面板不重叠。
